NetApp Cloud ONTAP and Cloud Manager Resources

NetApp® Cloud ONTAP is a software-only storage appliance that runs the NetApp clustered Data ONTAP® storage operating system. It provides secure, proven NFS, CIFS, and iSCSI data management for the cloud. You must use OnCommand® Cloud Manager to launch Cloud ONTAP as an EC2 instance in AWS. Use the resources on this page to deploy and use both Cloud ONTAP and Cloud Manager.

The resources on this page are for Cloud Manager 2.1 and Cloud ONTAP 8.3.2 RC1, which are the most recent releases.
